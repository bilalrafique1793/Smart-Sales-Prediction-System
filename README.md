# Smart Sales Prediction System

## UI Module

This project now includes a modern Streamlit interface for the Smart Sales Prediction System.

## What it includes

- A sidebar-based navigation with Home, Prediction, and Dashboard pages
- A polished landing page with objective, problem statement, and workflow
- A prediction form for TV, radio, newspaper budgets, and region
- A dashboard with dataset preview, descriptive statistics, EDA visuals, and model accuracy
- A shared footer with project metadata

## Files

- app.py - main Streamlit application entry point
- pages/ - reusable page modules and shared helpers
- assets/style.css - custom styling for the UI
- requirements.txt - Python dependencies for the app

## Run the app

```bash
streamlit run app.py
```

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```
